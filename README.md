qp是什么?
--------------
如果你也是做棋牌的程序猿(作者已弃坑)那么可以了解一下这个项目，本项目增加了Fiber，解决nodejs的各种回掉函数套用难读的问题

cocos团队努力的在推广他们的cocos creator，不过感觉现在已经算是个棋牌引擎了(滑稽)

想重新做一个nodejs的服务器以及对应ccc的客户端 单纯为了自己爽一下吧

centos6 centos7均可部署 centos7上需要自己安装mysql或者使用mariadb来代替mysql

本项目数据落硬盘主要用redis，并不打算重度使用mysql，但mysql是必不可少的，因为日志统计之类的信息记录后台统计工程师还是更熟悉一些。
