qp是什么?
--------------
cocos团队努力的在推广他们的cocos creator

因此想做一个nodejs的服务器以及对应的ccc的客户端

centos6 centos7均可部署 只是在centos7上需要自己安装mysql或者使用mariadb来代替mysql

不过说实话也不打算重度用mysql，最多也就是进行日志统计之类的信息记录，方便后台工程师使用
